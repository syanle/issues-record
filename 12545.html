ok了，看到英文就害怕的原因啦，还是自己解决的：
一、安装gevent 1.0
https://pypi.python.org/pypi/gevent#downloads
cd libev/
 ./configure
cd ..
sudo python setup.py install

二、安装dnslib
https://pypi.python.org/pypi/dnslib
sudo python setup.py install

三、运行Goagent
sudo goagent-osx.command
问题解决了
