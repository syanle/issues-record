GSnova在Heroku和Openshift上部署成功了也是慢得不像样。
用GSnova的GAE模式吧，部署方法见官网，Windows最新版应该是gsnova_0.23.1_windows_386。
最后，要把hosts.conf文件里GoogleHttpsIP= 和GoogleHttps= 的IP换成可用的IP，可以使用跟GoAgent一样的IP。
实测比GoAgent还稳定。
