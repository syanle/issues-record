请这样操作。
1. 先运行local\taskbar.exe
2. 讲下面两行代码加入到server\uploader.bat开头
set HTTP_PROXY=http://127.0.0.1:8087
set HTTPS_PROXY=http://127.0.0.1:8087
3. 双击uploader.bat上传，需要输入appid和你的用户名密码
4. 上传成功后，修改proxy.ini的appid, 重启taskbar.exe
