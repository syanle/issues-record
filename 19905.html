版本说明
2.x 版本

分支：https://github.com/holmium/dnsforwarder/tree/master

可执行文件：https://github.com/holmium/dnsforwarder/tree/master/codeblocks_win/bin

配置文件：https://github.com/holmium/dnsforwarder/blob/master/default.config

5.x 版本

分支：https://github.com/holmium/dnsforwarder/tree/5

可执行文件：https://github.com/holmium/dnsforwarder/tree/5/codeblocks_win/bin

配置文件：https://github.com/holmium/dnsforwarder/blob/5/default.config

2.x 版本和 5.x 版本的区别

5.x 版本现仅供测试。

因为 2.x 版本的性能瓶颈在于程序的架构，所以在 5.x 版本中改进了程序的架构。如果您需要较大量的查询，比如为整个局域网提供防污染能力，您可以试用一下 5.x 版本。

Windows
如果要使用 2.x 版本可以到 http://micasmica.blogspot.com/2011/08/dns.html 下载已打包好的程序，或者：

1.下载您想要使用版本的可执行文件

2.下载相应版本的配置文件

将配置文件和可执行文件放到一个文件夹中并把配置文件重命名为 dnsforwarder.config。

3.对配置文件进行适当的修改，如果不知道如何修改，则跳过此步。

4.运行 dnsforwarder.exe

5.修改系统 DNS 为 127.0.0.1

Linux
1.下载您想要使用版本的源代码

2.编译和安装

编译选项请参考 https://github.com/holmium/dnsforwarder/wiki/%E7%BC%96%E8%AF%91。

3.用 root 权限运行 “dnsforwarder -p”，下载相应版本的配置文件到提示的文件夹中并重命名为 “config”。

4.运行 “dnsforwarder”

5.修改系统的 DNS。

防污染原理
有两种防污染方式。

防止污染方式一

此为默认的方式（已写到初始配置文件中），原理是通过国外 DNS 服务器查询（UDP 协议），并过滤伪造的 DNS 包。

配置文件写法：

# 务必将主服务器协议设为 UDP
PrimaryServer UDP

# UDP 服务器中务必有一个国外服务器
UDPServer 8.8.8.8,114.114.115.115

# 建议开启并发查询
ParallelQuery true

# 开启 UDP 防污染
UDPAntiPollution true

# 请在 `UDPBlock_IP' 选项中添加可能的伪造的 IP 地址

# 请将 `GfwList' 置空
GfwList
防止污染方式二

原理是没有污染的域名通过 UDP 协议查询，有污染的域名通过 TCP 协议来避免污染。

配置文件写法：

PrimaryServer UDP

# 请使用国外的 TCP 服务器
TCPServer 8.8.4.4,8.8.8.8

# 建议使用国内的 UDP 服务器
UDPServer 114.114.114.114,114.114.115.115

# 建议关闭 UDP 防污染
UDPAntiPollution false

# 加载 GFW List
GfwList https://autoproxy-gfwlist.googlecode.com/svn/trunk/gfwlist.txt〗
