#3 warmingf...@gmail.com 
# 2 意味着此时 GoAgent 的 GAE 代理不依赖本地 DNS 解析了，所有的 DNS 解析在美国本土完成。

做了个对比实验：

1. 使用 GAE 代理

	1. 彻底关闭本机的 DNS 解析（即在网络设置中心那里设置所有 DNS 地址为任何一个无效的 IP，此时本地 DNS 解析途径已封死）
	2. 打开 http://www.bilibili.com/video/tech-popular-science-1.html 发现浏览器状态栏显示瞬间完成了 DNS 解析并加载了网页

2. 使用 Shadowsocks 代理

	1. 同上
	2. 打开上述网址发现浏览器卡在了 DNS 解析，最终打开网页失败

结论：GoAgent 使用了远程 DNS 解析不依赖本地，Shadowsocks 依赖本地。
