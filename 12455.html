#16 zhaohao...@gmail.com 
I'm using archlinux and have the same problem. I've alse installed python2-gevent. I found that, when I ping 8.8.8.8|8.8.4.4|114.114.114.114|114.114.115.115, there's no response. However, I have anothor DNS server 192.168.x.x in my LAN. So I added 192.168.x.x into the list of servers in [dns], and set enable=1. And the problen is solved.

I found my DNS server by executing "nslookup google.com". So if you can reach google.com but have the same problem, you can try this solution. "nslookup" is in extra/dnsutils.
