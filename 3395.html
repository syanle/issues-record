Reported by kfc...@gmail.com, Apr 21, 2012 
A 机器是 Linux，一直开机。我在上面运行 goagent，将其 CA.crt 导入到了 B 机器（Windows，工作机）中。

访问 Google+ 等一切正常。但是有一些其他的网站，如 https://killwall.com，https://www.amazon.com 等，会提示：

该服务器的安全证书尚未生效！
You attempted to reach www.amazon.com, but the server presented a certificate that is not yet valid. No information is available to indicate whether that certificate can be trusted. Google Chrome cannot reliably guarantee that you are communicating with www.amazon.com and not an attacker. Your computer's clock is currently set to 2012年4月22日星期日下午2:48:11. Does that look right? If not, you should correct any error the refresh this page.
您不应再继续，尤其是如果您以前从未在此网站看到这一警告信息，则更不应继续操作。

点确认是可以继续，但每次出现这个提示都不爽。

请问应如何解决呢？谢谢！
