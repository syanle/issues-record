对于长期受DNS污染困扰的人来说（什么是DNS污染具体Google）

新版 GoAgent 的 DNS 代理功能是非常棒的解决方案。

[dns]
enable = 1    ; 开启
listen = 127.0.0.1:53   ; 监听地址与端口
remote = 8.8.8.8  ; 代访问的 DNS 服务器

DNS 服务器填为 127.0.0.1，

如果还想为局域网或者公网提供代理服务的话，

设置 listen = <Current IP>:53 就是网卡设置什么IP就填什么IP

看下实际效果 e.x. 以 Google DNS 为例。
