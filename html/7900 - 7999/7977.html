1.问题描述

* win xp已经成功使用GAE实现翻墙

* 和windows用的同一个APPID出现此问题(不是流量不够)
* Linux下运行upload时不会像windows提示完appid后还要输入mail和pwd.

* 用新注册APPID(没有在windows下成功使用过), 直接提示appid不存在,但已经验证了手机短信,并登陆在google上是成功看到有表包含appid,title等注册信息. 但在windows下都可以顺利upload,只有在多个appid|appid的情况下解析了第一个,第二个才会提示不存在此id,但分开用是好的.

* VMware 9.0 ubuntu 11.1 Python 2.7.2+ gevent pyssl 为主页推荐链接的版本
* windows xp pro sp3下安装了VMware9.0 + ubuntu 11.1
