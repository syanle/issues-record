国内许多部门，尤其是学术、科研、外贸、外事等部门，没有谷歌，真的不行，因为他们需要的许多资料，从百度检索不到。

但许多人只知道谷歌的一个网址 google.cn，不知道谷歌其实有 20 万个 ip 地址，从这 20 万个 ip 地址中随便拿出一个，即可检索资料，效果跟 google.cn 一模一样。

懂技术的网友，可以使用 nslookup 命令查找这 20 万个 ip 地址。不懂技术的网友，可以使用下列网站，查找这 20 万个 ip 地址中可用的 ip 地址。

http://www.jscan.co.nr

查找方法很简单：按一下 scan 按钮，等两、三分钟，即可看到一批 ip 地址，其中红的是不能用的，蓝的是能用的。（万一没有蓝的，可以按一下 reload 按钮，重复上面的操作。）

上面提到的这 20 万个 ip 地址，是谷歌公司自己独有的，此外，谷歌在世界许多国家还有备用 ip 地址（这些备用 ip 地址的正式名称叫 google global cache，谷歌全球缓存），具体数量不详，估计也有几十万个。

谷歌拥有这么多 ip 地址的原因大致有两个，一是负载分流，让各国用户访问离自己最近的主机；而是服务分配，让不同的服务使用不同的 ip 地址。

祝您找到适合自己的 ip 地址！
