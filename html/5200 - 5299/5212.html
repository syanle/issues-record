#5 Taiyang....@gmail.com 
Ubuntu 下 sudo apt-get install libevent-dev; sudo easy_install gevent 就可以了。

Mac 下面需要用 macports/homebrew 安装libevent，比较麻烦，请问能否将 gevent+libevent binary 集成到下载包里？对新手来说安装这些很不方便。

mac 下可以到http://www.gevent.org/下载gevent源代码包到本地，然后解压后，要先运行下其中的

python fetch_libevent.py

然后 python setup.py build
python setup.py install

然后，就没有然后了。
