#6 Chenxing...@gmail.com 
其实尝试了半天，默认的Ubuntu 12.04下顺序应当是这样：
libevent-dev --> python-setuptools --> easy_install greenlet --> easy-install gevent

* 但其实有更简单粗暴的方法！！

直接在终端下
$ sudo apt-get install python-gevent
然后就完事了。
然后进入python交互终端：
$ python
>>> import gevent
只要不报错，然后<ctrl>+D退出然后正常安装即可。
