使用方法：保存成goagent.bat放置到goagent.exe目录运行即可。
——————————————————————————————————

@echo off

title 快速配置IE代理
:start
cls
echo.
echo         请选择需要进行的设置：
echo.
echo         1,使用代理     2,停用代理      3,删除IE临时文件       Q,退出
echo.
set /p choice=      请作出选择:
if /i "%choice%"=="1" goto start1
if /i "%choice%"=="2" goto enbal
if /i "%choice%"=="3" goto deltemp
if /i "%choice%"=="Q" goto end
goto start

:start1
echo	启动goagent
@taskkill /f /IM goagent.exe 1>nul 2>nul
start "" "goagent.exe"
echo	启动IE代理
reg add "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings" /v ProxyServer /d "127.0.0.1:8087" /f
reg add "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings" /v ProxyEnable /t REG_DWORD /d 1 /f
echo     目前设置为使用代理,任意键返回.
pause>nul
goto start


:enbal
echo 停止IE代理设置
reg add "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings" /v ProxyEnable /t REG_DWORD /d 0 /f
echo 停止goagent
@taskkill /f /IM goagent.exe
echo.
echo     目前设置为停用代理,任意键返回.
pause>nul
goto start

:deltemp
@ECHO OFF
echo 正在清除临时文件，请稍等...... 
del /f /s /q %systemdrive%\*.tmp 
del /f /s /q %systemdrive%\*._mp 
del /f /s /q %systemdrive%\*.log 
del /f /s /q %systemdrive%\*.gid 
del /f /s /q %systemdrive%\*.chk 
del /f /s /q %systemdrive%\*.old 
del /f /s /q %systemdrive%\recycled\*.* 
del /f /s /q %windir%\*.bak 
del /f /s /q %windir%\prefetch\*.* 
rd /s /q %windir%\temp & md %windir%\temp 
del /f /q %userprofile%\cookies\*.* 
del /f /q %userprofile%\recent\*.* 
del /f /s /q "%userprofile%\Local Settings\Temporary Internet Files\*.*" 
del /f /s /q "%userprofile%\Local Settings\Temp\*.*" 
del /f /s /q "%userprofile%\recent\*.*" 
pause
goto start

:end
exit

再添加mac os系统下一键启停goagent和自动起停代理的脚本设置：
打开终端，执行：
sudo -s
vi  /usr/bin/myproxy
然后添加如下内容：
#!/bin/bash
# 使用方法
# 运行该脚本，即开启代理。Ctrl-c 中止该脚本，即关闭代理。

disable_proxy()
{
networksetup -setwebproxystate Wi-Fi off
networksetup -setwebproxystate Ethernet off
echo "Web proxy disabled."
}
trap disable_proxy INT

networksetup -setwebproxy Wi-Fi 127.0.0.1 8087
networksetup -setwebproxy Ethernet 127.0.0.1 8087 
networksetup -setwebproxystate Wi-Fi on
networksetup -setwebproxystate Ethernet on
echo "Web proxy enabled."
echo "Starting goagent..."
killall python
/usr/bin/python /Users/goagent/local/proxy.py
killall Terminal



完成并保存后，再给该文件添加可执行权限，并把下载的最新版goagent解压后放到/Users/目录，执行myproxy脚本即可
如果想更方便，可以自己实用AppleScript制作一个应用来调用该脚本：
打开"实用工具"－－“AppleScript编辑器”输入如下内容：
tell application "Terminal"
	activate
	do script with command "/usr/bin/myproxy"
end tell
然后点“文件”－－“储存为”
在存储为：后面填入goagent，位置那里选择“应用程序”；文件格式选择“应用程序”，然后存储即可

去打开你的“finder”－－“应用程序”里找到goagent，点击打开即可看到goagent熟悉的界面，打开该脚本，会自动启动goagent并设置代理，使用ctl ＋ c可以关闭goagent并自动关闭代理设置。
