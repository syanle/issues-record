#9 ahx...@gmail.com 
我总结了规律

如果设置了 regions =  选项，它会在打开一个网站前用本地电脑上的解析服务（本地没有 DNS 就用内置的）， 如果成功判断 regiions 是中国日本等则用已经本机解析好的 IP 直连。如果本机返回的是 GoAgent 内置的黑名单或返回的 IP 不是中国日本等的就用代理那边的 DNS 解析并连接，另外 .ini 配置文件里的 DNS 模块设置 enable = 1 效果就和 DNSForward 一样了，二者不可共存，DNSForward 速度优于 GoAgent 内置的。
Aug 21, 2014
#10 ahx...@gmail.com 
如果出现黄字，很可能是因为 GoAgent 获取本地返回的 DNS 结果失败，没有从 127.0.0.1:53 获取结果，所有用了 GoAgent 内置的。
