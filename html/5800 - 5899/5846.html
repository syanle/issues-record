#2 scy...@gmail.com 
那是因为我镇的人民公仆为了洗赢党的18摸，被功夫网给XXOO了，解决办法 

1.修改配置文件  proxy.ini
节点 [gae]
把 profile = google_hk  修改为 profile=google_cn

2.配置所有连接方式为https，（为什么你懂的，GFW也是很有实力的）
[google_cn]
mode = https

[google_hk]
mode = https

OK了，就这样，可以用了，还是跟以前的一样使用就可以了。
