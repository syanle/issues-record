#2 anesli...@gmail.com 
首先需要确认你是否能够连接到ipv6网络，
如果你是纯6网络，依然不能正常访问，可尝试：1.修改墙外的dns。2.配置hosts列表。3.使用dns crypt。
如果你使用双栈网络，尝试在命令行输入如下命令，提高ipv6访问优先级
netsh int ipv6 set prefix ::1/128 50 0
netsh int ipv6 set prefix ::/0 40 1
netsh int ipv6 set prefix 2002::/16 30 1
netsh int ipv6 set prefix ::/96 20 3
netsh int ipv6 set prefix ::ffff:0:0/96 10 4
netsh int ipv6 set prefix 2001::/32 5 1

#10 anesli...@gmail.com 
To #6
这些设置的意义就是修改路由优先级，在双栈网络环境中，优先使用ipv6访问。
提高优先级后，设置一个支持解析ipv6的dns，对于部署了ipv6的网站，如google系列网站，youtube，facebook等，一般情况不需使用goagent就可访问。
