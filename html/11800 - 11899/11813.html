#2 thatsrea...@gmail.com 
问题解决，速度超快。不过用pip install --upgrade crypto不行，用下载的pycrypto也不知如果安装。最后是用 sudo easy_install PyCrypto才装上的，必须在翻墙状态下才能完成。好象之前还安装了gcc,不知是否必要。
Dec 11, 2013
#3 thatsrea...@gmail.com 
1. 安装http://osxdaily.com/2012/07/06/install-gcc-without-xcode-in-mac-os-x/
2. 打开Terminal，键入：sudo easy_install PyCrypto
3. 此事应有网络联接，这条命令会直接去网找PyCrypto下载并安装。完了就一切正常了。
