#67 jerry011...@gmail.com 
哈哈，成功了，小白做的小白教程来了。各位大神看看有错误没？
先下载最新版的goagent
1.去https://github.com/moonshawdo/checkgoogleip 下载ZIP，把解压得到的checkip.py等等一堆文件复制到goagent的local目录。
2.运行checkip.bat,然后打开ip_tmpno，复制里面的IP。
4.去https://github.com/azzvx/gogotester/releases/下载 Source code (zip)，把解压出来的GoGo Tester复制到goagent的local目录，运行。
5.对着GoGo Tester窗口里，右键，导入，剪贴板存在的IP，粘贴ip_tmpno的一些IP，全选，右键，应用，应用到用户配置文件。
6.重启goagent，Enjoy.
