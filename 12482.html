#12 wwqgtxx 
generate_RSA.bat只是用来生成一对公私钥，本身并不会流出到任何文件中，需要自行创建key_config.py把公私钥放进去

公钥的格式是这样的
-----BEGIN PUBLIC KEY-----
..........
-----END PUBLIC KEY-----
私钥的格式是这样的
-----BEGIN RSA PRIVATE KEY-----
..........
-----END RSA PRIVATE KEY-----
分别填入/local/key_config.py
#!/usr/bin/env python
# coding:utf-8

__RSA_KEY__ = '''
-----BEGIN PUBLIC KEY-----
..........
-----END PUBLIC KEY-----
'''

和/server/gae/key_config.py这个文件中

#!/usr/bin/env python
# coding:utf-8

__RSA_KEY__ = '''
-----BEGIN RSA PRIVATE KEY-----
..........
-----END RSA PRIVATE KEY-----
'''
