貌似採用預設值的GoAgent等是不能上傳UTB影片的，只能觀看影片。因為上傳影片時會產生大量網路流量，伺服器可能會認為這是屬於惡意行為。不曉得採用全局代理或者是調整下GoAgent的setting能否解決這個問題。

應急：http://www.iqlinkus.net/home?referral=71E719CF890F35714C6940C44CF58D36
如果你實在急於上傳影片的話，試試上面這個連結，註冊一個VPN帳號（免費流量500MB/Month)，然後連線位於美國的server，就可以上傳兩三百MB的影片。影片先轉檔成MP4格式，這樣容量較小，建議用MediaCoder，這個轉檔好，按照預設值那樣轉檔成MP4，影片品質跟母影片差不了多少，最重要的是可以縮小檔案大小。http://blog.xuite.net/yh96301/blog/72795921-%E5%85%8D%E8%B2%BB%E5%BD%B1%E9%9F%B3%E8%BD%89%E6%8F%9B%E8%BB%9F%E9%AB%94MediaCoder（用GoAgent開啟這個網頁）

在花费了一段时间进行调整后，发现对于3.1.5版本的GoAgent，如果需要上传YouTube影片，请在proxy.ini中，找到以下字段：
; https?://www\.youtube\.com/watch = google_hk
; ttps?://.+\.youtube\.com/.*yt_live = google_hk
; https?://.+\.c\.youtube\.com/.*yt_live = google_hk
; https?://.+\.c\.youtube\.com/liveplay = google_hk
; https?://.+\.youtube\.com/live = google_hk
; https?://.+\.youtube\.com/upload = google_hk
将前面的";"删除。
另外，上传需要在地址前加上"https://"。
