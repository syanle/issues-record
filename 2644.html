#1 simple59...@gmail.com 
在开始菜单里面有一个开机启动程序那一项，添加一个goagent的启动程序就可以了。
添加的内容可以这么写 python /goagent所在文件夹路径/goagent/local/proxy.py，python前面好像还有一个terminal选项来着，具体有点记不清楚了，查一下以后在告诉你
Feb 27, 2012
#3 sun...@gmail.com 
我前兩天查過，假設你的 goagent 位於 ~/ 之下，用文本編輯器編輯 ~/.profile 文件，在最下面加入：
# goagent 這一行沒有實際作用，只是標註
python ~/goagent/local/proxy.py > /dec/null 2>&1 &
cd ~
一共三行，加入之後，保存（確保你有權限修改）再重啟電腦即可。
