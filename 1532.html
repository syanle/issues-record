....呵呵　既然来了就帮ＬＺ解决下问题吧　。

wallproxy\local\cert　　到这个目录下找到_ca.crt证书文件　，双击安装证书，选择第二项　将所有的证书放入下列储存区　点击浏览　选择受信任的根证书颁发机构。重启浏览器之后应该就可以了。如果不行的话就在浏览器里找到证书这一项然后按上面的操作导入　。

#21 mzh...@gmail.com 
在Ubuntu之类的系统中可以这样
sudo apt-get install libnss3-tools -y


然后导入证书：


certutil -d sql:$HOME/.pki/nssdb -A -t TC -n "goagent" -i /path/to/goagent/local/CA.crt
