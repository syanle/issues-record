#6 Kishen...@gmail.com 
运行环境是搬瓦工25一年的VPS，系统centos x86

说下我的操作经历：

1.首先下载了官方的goagent https://github.com/goagent/goagent

2.上传go VPS文件，因为搬瓦工默认SSH的端口是随机生成的，而goagent vps默认端口是22，所以要改server文件夹下的uploadvps.py文件。鼠标右键uploadvps.py，选择用记事本打开该文件，
有点乱，但是22这个数字还是比较好找的：

 client.connect(host, 22, username, password)
  
把这段里的22，改成你搬瓦工随机生成的端口，例如：

client.connect(host,你的端口, username, password)

然后输入你VPS的IP，root用户名，及密码，上传完成。

3. ssh进入到VPS server。执行如下操作。

sudo yum install gcc libffi-devel python-devel openssl-devel python-pip

（用centos minimal的时候执行这个：yum install gcc libffi-devel python-devel openssl-devel python-pip）

4.然后执行 easy_install dnslib。结果返回命令：

bash: easy_install: command not found

这下坑爹了，貌似是默认系统没有安装easy_install

看一看python版本

python -V
Python 2.6

我去！好原始啊。赶紧先升级Python吧。到官网看看，那就2.7.8吧

wget https://www.python.org/ftp/python/2.7.8/Python-2.7.8.tar.xz

tar -xvf Python-2.7.8.tar.xz

cd Python-2.7.8

./configure

make

su

make install

安装好了再看一看python版本，2.7.8，妥妥的！下面赶紧下载easy_install

执行一下：

wget https://bootstrap.pypa.io/ez_setup.py -O - | python

dir

果然目录多了个setuptools-x.x.x.zip (这个版本随时更新，所以用XX代替）

然后解压：

unzip setuptools-x.x.x.zip 

(假如返回bash: unzip: command not found，则先则先执行yum install zip unzip -y 安装解压程序再解压）

先看一下文件夹名

dir 

看到解压的文件夹setuptools-x.x.x后，执行:

cd setuptools-x.x.x

进入文件夹执行：

python ez_setup.py

开始安装，如果出现finish，说明安装成功！

终于可以开始正题了:

依次执行：

easy_install dnslib
easy_install pycrypto
easy_install gevent
easy_install pyOpenSSL==0.13
cd /opt/goagent/vps
./goagentvps.sh start
tail -n 10 /opt/goagent/log/goagentvps.log

成功了

修改proxy.ini文件。文件在local目录里
  找到如下字符串
  
[vps]
enable = 0 （改为1）
listen = 127.0.0.1:8088
fetchserver = https://username:password@x.x.x.x/
这里的uservername就是VPS主机里的goagentvps.conf账号跟密码了。当然你可以自己编辑那个goagentvos.conf，在里面去添加你想添加的账号跟密码信息。修改完别忘记将脚本restart一下。
