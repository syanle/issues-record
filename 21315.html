Reported by warmingf...@gmail.com, Jun 6, 2015 
去年 GoGoTester 100 万 IP 地址库 （典型的如 210.242.125.* ) 被封，进而自助寻找 IP 地址的时代到来，那些由于各种原因没用上 checkgoogleip 的老基友走了，留下来的基友大多会了 checkgoogleip, 并逐渐成了经验丰富的老鸟。

因而这里就少了“自定义 IP 地址， checkgoogleip 和 GoGoTester ”相关的讨论，直至今日，相关话题重出江湖，标准着一大波新基友正在投入 GoAgent 的怀抱。更广泛的交流，更多的参与，思想碰撞此起彼伏。

                                                                           -----------序言
先下载最新的 GoAgent https://github.com/goagent/goagent/archive/3.0.zip

- 自定义 IP 的方法： IP 地址放到 proxy.ini 的 "=" 号后面，如

[iplist]
google_cn = 127.0.0.1|127.0.0.2
google_hk = 127.0.0.1|127.0.0.2

- 自动化扫未知 IP 段： IP 地址可由 checkgoogleip https://github.com/moonshawdo/checkgoogleip 扫描某一地区 IP 段获得，扫描结果（那几个 *.txt 文件）中凡是带 google, doubleclick 等的都是可用的，包括 *.googlevideo.com 。﻿这里提供了各国家与地区的 IP 段 http://ipblock.chacuo.net/ 。

- 手动优选 IP 地址（响应时间、带宽测试等）及扫描 IP 地址和 IP 地址段： GoGoTester https://code.google.com/p/gogo-tester/

其他问题：

问：怎样通过 checkgoogleip 获取 *.google.com, *.googlevideo.com, *.doubleclick.net 等之类的 IP 地址呢？

答： 以下几种方法

1. 由于 checkgoogleip 暂不支持输出 gvs IP 地址，可通过修改 checkgoogleip 代码使之能够输出 gvs IP  地址到 ip.txt


2. 使用文本编辑器 EverEdit , 以 "google" 之类的关键词“查找全部”，将查找结果复制并粘贴到一个新建文件窗口，以正则表达式 (\d+\.\d+\.\d+\.\d+).+ 替换为 $1 或通配符得到纯 IP 地址，即把 127.0.0.1 *.googlevideo.com 变成 127.0.0.1

3. 全选复制那些杂乱的文本 -> GoGoTester 内右键导入剪切板 IP 进行处理 -> 右键导出 IP 到剪切板得到标准格式的 IP 序列。

4. 其他高效的方法（如程序脚本）以类似 EverEdit 的原理分离出纯 IP 地址

5. 俺以上方法都不会，俺喜欢用敏锐的双眼和勤劳的单手把它们揪出来

问：上面那些话太麻烦，俺要用 XX-Net 这样的无障碍通道
答： https://github.com/XX-net/XX-Net
