#4 Zero.n.L...@gmail.com 
另外, 做了一个命令脚本, 可以方便一点.

@ECHO off
:begin
@CHOICE /C:sedq /N /M "(S)how state; (E)nable; (D)isable; (Q)uit"
IF ERRORLEVEL 4 GOTO end
IF ERRORLEVEL 3 GOTO disable
IF ERRORLEVEL 2 GOTO enable
IF ERRORLEVEL 1 GOTO show
rem GOTO begin
:show
netsh interface teredo show state
netsh interface 6to4 show state
netsh interface isatap show state
GOTO begin
:enable
netsh interface teredo set state default
netsh interface 6to4 set state default
netsh interface isatap set state default
GOTO begin
:disable
netsh interface teredo set state disable
netsh interface 6to4 set state disable
netsh interface isatap set state disable
GOTO begin
:end
rem @PAUSE
