if ((isPlainHostName(host) || dnsDomainIs(host, ".google.com")) && !localHostOrDomainIs(host, "talk.google.com") && !localHostOrDomainIs(host, "talkx.l.google.com")) return 'PROXY 127.0.0.1:8087';

意思是除了talk.google.com和talkx.l.google.com
其它所有的*.google.com都通过goagent代理访问
比如play.google.com，mail.google.com等
这条规则可以解决chrome浏览器启动同步的情况下
输出界面不断出现黄色5222端口超时提示的问题
