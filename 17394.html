Reported by wuyan12...@gmail.com, Sep 23, 2014 
  现在good ip越来越不好找了 大家都有感觉 就现在的情况看只能是每况愈下 所有ip被封估计就是时间问题
  这几天尝试了一个比较简单的办法 不用找ip也继续使用goagent
  其实就是简单的使用 前置代理
[proxy]
enable = 1（打开）
autodetect = 1
host = 10.64.1.63 (找到的https（ssl）代理地址)
port = 8080 (端口)
username =
password =
只要将网上找到的免费代理填上就行了 其他设置不用改（id还是要填的)
测试发现只要代理速度够快 完全可以无痛看油管1080p
代理必须满足两个条件 1 必须是墙外的代理 2 必须支持https（ssl）
根据经验除80端口外的代理 多数支持https（ssl）
3128端口很好 奇怪的本人这里居然被isp封掉此端口
免费的好代理不多 且需要一个一个试 比较麻烦 适合不怕折腾的朋友
这里提供两个找免费代理的地方
1 代理中国论坛 这里有人发免费代理 一次上千个 不过要自己测试
2 http://www.freeproxylists.net/zh/ 这个是google找到的 代理也不少 常更新 而且可以筛选代理
个人经验 分段下载参数[autorange] 对视频流畅度影响较大 可以根据代理速度适当调节

鉴于目前形式 希望作者能加强一下前置代理功能
比如可以使用多个前置代理均衡 让goagent自动测试判断调度前置代理
我用一个笨办法使用多条速度不太理想的代理实现流畅油管1080p 就是用一个叫MultiProxy的自动代理调度软件实现流量均衡
如果作者能让goagent自己可以调度前置代理就太好了
  
  最后 个人认为 goagent和他的作者将名垂青史
