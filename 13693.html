发issue前先在All issues中搜索是否已经有你要提交的问题。

我也是北京联通用户，发现最近很慢，原因是由于GFW对google hk的封锁，我已经解决了，发给大家参考。如下：

我用的是3.1.2-12，速度得到很大提升。直接在proxy.ini的google_hk,hosts=字段最后加入了www.google.com.tw，速度马上改善。通过google tw 解析后，速度恢复正常了
[google_hk]
mode = https
window = 4
hosts = www.google.com|mail.google.com|www.l.google.com|mail.l.google.com|www.google.com.hk|www.google.com.tw

#3 xin.cr...@gmail.com 
小白教程给不熟悉的同学。
1. 用记事本打开proxy.ini
..\goagent\local\proxy.ini


2. 找到 gae部分，检查 profile = google_hk
[gae]
appid = XXXXX
password = 
path = /2
profile = google_hk
crlf = 1
obfuscate = 0
validate = 0

3. 找到[google_hk]，hosts=字段最后加入了www.google.com.tw

[google_hk]
mode = https
window = 4
hosts = www.google.com|mail.google.com|www.l.google.com|mail.l.google.com|www.google.com.hk|www.google.com.tw

速度不理想的可以试试多加入几个未封锁的google 服务器，如www.google.com.jp
