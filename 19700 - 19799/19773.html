#4 5linh...@gmail.com 
lz小白
看你的ini文件设置：
‘’‘’
[listen]
ip = 127.0.0.1
‘’‘’
代表监听当前这个电脑，即本地
如果你的电脑有两个网卡，一个连接内网，一个连接公网
则
‘’‘’
[listen]
ip = 内网ip
‘’‘’
代表监听内网中的电脑，也就是说只有内网中的ip可以访问，本地也可访问
而
‘’‘’
[listen]
ip = 公网ip
‘’‘’
代表监听公网中的电脑，也就是说只有公网中的ip可以访问，内网也访问不了，但本地可以访问
而
‘’‘’
[listen]
ip = 0.0.0.0
‘’‘’
代表监听所有网络中的电脑
