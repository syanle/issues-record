此方法不是我原创的，已经有神大人解决了。需要在“sites =”后面添加的内容如下：
| .google.com|.googleusercontent.com|.googleapis.com|.google-analytics.com|.googlecode.com|.google.com.hk|.appspot.com|.android.com|.googlegroups.com|www.youtube.com/metadata_ajax|www.youtube.com/api/tag_suggest|www.youtube.com/api/upload_feedback|www.youtube.com/upload/|upload.youtube.com|www.youtube.com/gen|www.youtube.com/my_videos_upload


在proxy.ini中，找到所有sites=相关的配置行。
[google_cn]
mode = https
hosts =*
sites = *
forcehttps = *
withgae = *

[google_hk]
mode = https
hosts =*
sites = *
forcehttps = *
withgae = *

[crlf]
enable = 0
dns = 8.8.8.8
sites = *

把要添加的内容添加到[google_cn]、[google_hk]、[crlf]这三个配置项中的“sites =”配置行后面。如果不清楚可以直接下载我的配置文件(见附件)，覆盖你的文件就行。
