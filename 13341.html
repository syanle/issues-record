OK，找到解决办法了～

在wget.exe后面添加--secure-protocol=TLSv1 参数就可以了～～

请暂时无法更新的朋友，可以自行在wget.exe后添加该参数，或将wget那一行替换成下面内容：

@path %PATH%;%windir%;%windir%\system32
@netstat -an|find "LISTENING"|find ":8087" && set HTTP_PROXY=http://127.0.0.1:8087 && set HTTPS_PROXY=http://127.0.0.1:8087
@wget.exe -O goagent.zip --secure-protocol=TLSv1 --no-check-certificate https://nodeload.github.com/goagent/goagent/legacy.zip/3.0 ||echo %downloaderror%  &&pause &&exit
