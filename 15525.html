Reported by ahx...@gmail.com, Aug 3, 2014 
看视频下载等出现红字 502 是因为 AppID 对应的 GAE Server 流量被谷歌暂时限制，

重启 GoAgent 或 右键托盘 「重新载入」 立马红字消失，并继续缓冲是因为这样操作随机更换使用了另一个 AppID ，正好这个 AppID 此时没被流量限制，所以继续完美运行。我填了 25 个 AppID ，保证了每次随机到并更换的不会是上一个被临时屏蔽的 AppID

感觉可以在应用层面解决出现红字后立马重新载入等操作更换到另一个 AppID，这样就可以接触流量限制了。

希望英雄能加入这个功能。

但是重新载入更换 AppID 的时机要把握好，不然不支持断点续传的连接会失败，Youtube 会找不到视频。

希望英雄能加入一个 502 后重新更换 AppID 的选项。只是更换 AppID 的时机要刚好早于出现 502 才行。。。英雄辛苦了！！！
