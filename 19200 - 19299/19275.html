RT，使用最新的GoAgent，可以打开google.com.hk,youtube,facebook等网站，但是却打不开Android开发网（developer.android.com），请教下原因或解决办法？

浏览器版本：Firefox 34.05
GoAgent 版本：3.2.3
proxy.ini配置如下:

google_hk = www.google.com|mail.google.com|www.google.com.hk|www.google.com.tw|www.l.google.com|www2.l.google.com|ww    w3.l.google.com|www4.l.google.com|www5.l.google.com|mail.l.google.com|googleapis.l.google.com|googlecode.l.googleuse    rcontent.com|maps.l.google.com|code.l.google.com|cert-test.sandbox.google.com

[profile]
android.com = google_hk

#3 byhon...@gmail.com 
问题解决了，用pac规则导致android官网默认用DIRECT模式。
现在临时workaround是使用127.0.0.1：8087方式，至于是不是各bug，或是在线列表更新缓慢问题，待测～～

Change proxy.pac and add "android.com".

function FindProxyForURL(url, host) {
    var autoproxy = 'PROXY 127.0.0.1:8087';
    ......
    } else if (shExpMatch(host, '*.google*.*') ||
        ......
        dnsDomainIs(host, '.android.com') ||
        ......
        return autoproxy;
    }
}
